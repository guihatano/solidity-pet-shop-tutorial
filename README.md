# Pet shop

DApp using Truffle Suite

## Pre-requisites

1. Install [NodeJS](https://nodejs.org/en/)
2. Install `Truffle`

   `npm install -g truffle`
3. Install [Metamask](https://metamask.io/)
4. Install [Ganache](https://truffleframework.com/ganache)

## Run server

```shell
npm run dev
```
